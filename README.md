# Gregtech Skybound: GCYL Edition
GS:GCYL is a lightweight Skyblock modpack focused around GTCEu + GCYL and Ex Nihilo with some QoL mods

## Building
You will need to have python3 and python requests (`pip3 install requests`) installed.
To build run `python build/main.py` in command line.
